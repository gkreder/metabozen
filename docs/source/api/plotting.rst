Visualization Functions
-------------------------

The plotting module provides various visualization functions for metabolomics data:

- Volcano plots for statistical analysis
- PCA plots for sample clustering
- Heatmaps for feature intensity visualization
- Chromatogram plots for individual features
- Mass spectra visualization
